# JobRadar 🎯

**JobRadar** is a modern, privacy-centric job application tracking system built as a Single Page Application (SPA). Designed for the organized job seeker, it provides a streamlined environment to log and monitor applications without the need for cloud databases or third-party accounts.

## 📖 Project Overview

The core objective of JobRadar is to eliminate the "black hole" of job hunting. By providing a distraction-free interface, users can manage their career pipeline with speed and security.

Unlike traditional trackers, **JobRadar operates on a "Zero-Server" philosophy**, meaning your sensitive career data never leaves your machine.

---

## ✨ Key Features

* **100% Local Privacy:** All data is persisted via the browser's `LocalStorage`. No APIs, no databases, and no third-party tracking.
* **Responsive Shell Architecture:** A "Glassmorphism" styled navigation bar provides a consistent experience across all views.
* **Mobile-First Design:** Optimized for on-the-go tracking with a custom collapsible hamburger menu and touch-friendly targets.
* **Dashboard Analytics:** A centralized hub to view the status of all active and past applications.
* **Modern UI/UX:** Built with CSS variables for easy theming and backdrop filters for a premium, frosted-glass aesthetic.

---

## 🛠️ Technical Stack

* **Framework:** [Angular](https://angular.io/) (Component-based architecture)
* **UI Framework:** [Bootstrap 5](https://getbootstrap.com/) (Grid system & Utility classes)
* **Styling:** Custom CSS with CSS Custom Properties (Variables)
* **State Management:** Browser LocalStorage API
* **Routing:** Angular Router for seamless SPA navigation

## 🚀 Installation & Setup

To get a local copy up and running, follow these simple steps:

### Prerequisites

* Node.js (v18 or higher)
* Angular CLI (`npm install -g @angular/cli`)

### Setup Steps

1. **Clone the repo**
```bash
git clone https://github.com/yourusername/jobradar.git

```


2. **Install NPM packages**
```bash
npm install

```


3. **Run the Application**
```bash
ng serve

```


4. **View in Browser**
Open your browser and navigate to `http://localhost:4200/`
